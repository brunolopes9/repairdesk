# Script de vídeo demo — 90 segundos

Data: 2026-05-17
Propósito: vídeo único de 90s que mostra **a diferenciação real** do RepairDesk. Para usar em:
- Landing page (`lopestech.pt/repairdesk`)
- Cold outreach a oficinas (DM, email)
- Pitch a sócios/investidores se aparecer
- LinkedIn / Instagram da LopesTech

---

## Princípios

1. **Mostra, não digas.** Cada feature aparece no ecrã.
2. **Cliente final é o herói** — não a oficina. Tudo é "como a tua loja parece profissional ao cliente".
3. **PT-PT autêntico.** Sem inglês desnecessário. Sem corporativo.
4. **0 emojis falados, 0 buzzwords** ("revolucionário", "IA", "disruptive"). Concreto.
5. **Sem música épica.** Voz natural. Música ambiente discreta opcional.
6. **Termina com call-to-action concreto.** Não "saiba mais".

## Notas técnicas para gravação

### Ferramenta recomendada
- **Loom** (mais simples, gratuito até 5min) — gravas o ecrã + voz + webcam pequena no canto. Bom para protótipo.
- **OBS Studio** (gratuito, mais controlo) — recomendado para versão final
- **ScreenFlow** (Mac) ou **Camtasia** (Win) — pagos, edição mais polida

### Resolução
- 1080p mínimo (1920×1080)
- 30fps suficiente
- Áudio: micro decente (não built-in do laptop). USB tipo Blue Yeti ou Samson Q2U (€60-100).

### Setup antes de gravar
- Tenant demo com **dados credíveis** (Maria Silva, iPhone 13 Pro, valores reais €)
- Browser em **incógnito** (sem extensões, sem bookmarks visíveis)
- Resolução do browser: 1440×900 (deixa margem para texto sobreposto em edição)
- **Limpa** todas as notificações do sistema (mute do telefone, fecha Slack/WhatsApp Desktop)

### Erros a evitar
- ❌ Música tipo "epic startup video" — distrai e parece fake
- ❌ Falar demasiado depressa para encaixar — corta uma cena se necessário
- ❌ Mostrar a UI a meio de loading state — espera tudo carregado
- ❌ Mock data com nomes tipo "Test User" / "Foo Bar" — usa nomes PT credíveis
- ❌ Mostrar erros, console aberto, dev tools, ou edge cases — é demo, não debug session
- ❌ Voz monocórdica — varia tom. Tu próprio falas com energia em conversas reais — repete isso aqui.

Cortes:
1. **Hook (3s):** "A tua oficina merece melhor que Excel."
2. **Mostra portal cliente em mobile (10s):** Maria abre QR, vê timeline, fotos, health score
3. **CTA (5s):** "Beta a abrir Junho 2026. lopestech.pt/repairdesk."
