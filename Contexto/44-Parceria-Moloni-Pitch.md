## Porquê negociar parceria

**Argumento económico para o Moloni:**

Cada tenant que o RepairDesk traga subscreve Moloni Flex (€10.90/mês) ou Pro (€15.90/mês).
Para 30 tenants em 12 meses = **€3.924-5.724/ano de receita recorrente** que o Moloni NÃO teria sem o RepairDesk como funil.

**Argumento para nós:**

1. **Desconto vitalício 20-30%** para tenants RepairDesk em Flex → diferenciador de preço
2. **Referral revenue share** 10-20% do primeiro ano → revenue extra
3. **Marketplace listing destacado** → tráfego orgânico
4. **Co-marketing** → credibilidade do Moloni transfere-se para o RepairDesk
5. **Acesso prioritário a suporte técnico** → resolve bugs Moloni mais depressa

## Estratégia de negociação

### O que pedir AGORA (na primeira conversa)

1. **Desconto 20-30% nos planos Flex/Pro** para tenants que cheguem via RepairDesk
   - Argumento: cada um traz €130-190/ano de receita nova ao Moloni
   - Comparação: parcerias Visma/Moloni semelhantes em outros países

2. **Referral code / affiliate** — 10-15% do primeiro ano de cada subscrição via RepairDesk
   - Standard em SaaS partnerships
   - Fácil de implementar (código tracking na URL de signup)

3. **Trial Flex 60 dias** (vs 30 standard) para oficinas que vêm via RepairDesk
   - Reduz fricção, aumenta conversão
   - Custo marginal para Moloni: zero (já dão 30 dias)

4. **Marketplace listing destacado** no Moloni Marketplace
   - Tráfego orgânico
   - Credibilidade

5. **Badge "Parceiro Oficial Moloni"** no nosso site
   - Aumenta conversão dos nossos visitantes

### O que NÃO pedir já (esperar tracção)

- Investimento da Visma/Moloni
- Acordo de exclusividade (mau para nós)
- Logo deles destacado no nosso onboarding (depende do contrato)
- Acesso a dados agregados dos seus clientes (legal sensível)

### Triggers para revisitar termos

- **5 tenants pagantes** via RepairDesk → renegociar % referral para cima
- **20 tenants** → propor case study + webinar conjunto
- **50 tenants** → propor partnership tier superior (acesso a beta features Moloni)
- **100+ tenants** → conversa sobre integração mais profunda (e-Fatura embedded, etc.)
